Use Custom Code Adder by AceApp Studios to safely add your custom code to your site, instead of editing your core files.

== Description ==

Easily add HTML, PHP, JavaScript, or almost any other code to your Wordpress site. Codes are stored safely in the database so that when switching themes or updating the plugin your custom code is not lost.

== Installation ==

1. Upload `ZIP` file in the 'Upload Plugins' menu in WordPress
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit the plugin settings page to add your custom codes
